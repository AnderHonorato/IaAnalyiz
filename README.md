# IaAnalyiz - Projeto Bot ML

Este é o repositório principal do projeto **IaAnalyiz**, contendo tanto a estrutura do Frontend como a do Backend.

## 🚀 Tecnologias Utilizadas

* **Frontend:** (Ex: React, Next.js, HTML/CSS, etc. - preenche com as tuas tecnologias)
* **Backend:** (Ex: Node.js, Express, Python, etc. - preenche com as tuas tecnologias)

## 📁 Estrutura do Projeto

O projeto está dividido em duas partes principais:
* `/frontend` - Contém todo o código da interface de utilizador.
* `/backend` - Contém a lógica do servidor, APIs e ligação à base de dados.
